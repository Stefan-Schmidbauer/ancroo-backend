Rich Text (contenteditable)

This is a rich text area. You can select parts of this text and test workflows like:

  • Replace Selection — replaces what you selected
  • Insert Before — adds text before your selection
  • Insert After — adds text after your selection

Try selecting just one word or an entire paragraph to see how each output action behaves.

How to use

  1. Select text in any field above
  2. Open the Ancroo side panel (or use a hotkey)
  3. Choose a text transformation workflow and execute it
  4. The result depends on the workflow's output action: replace, insert before/after, clipboard, or side panel only

Manual Input workflows don't need a selection — open the side panel and type your text directly.
